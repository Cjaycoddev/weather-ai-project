Weather AI Dashboard

A modern weather dashboard built with React and Vite that integrates the Weather AI API to provide real-time weather information, location-based forecasts, city search, and dynamic weather visualizations.

Project Features

Real-time weather data
Automatic location detection
City search functionality
7-day weather forecast
Dynamic backgrounds based on weather conditions
Country flag support
Search history using local storage
Responsive design for desktop and mobile devices

Technology Stack

React
Vite
JavaScript
Tailwind CSS
Axios
Weather AI API

Installation Instructions

Clone the repository

git clone

Navigate into the project folder

cd weather-ai-dashboard

Install dependencies

npm install

Create a .env file in the project root and add your API key

VITE_WEATHER_AI_KEY=your_api_key

Start the development server

npm run dev

Production Build

To generate a production build:

npm run build
